# yup_study
This project investigates the unconventional market performance and successful user acquisition strategies of the Yup visa card in Indonesia.

# 印尼Yup信用卡市场分析项目

## 项目概述

这是一个**完整的个人商业分析项目**，旨在理解印尼金融科技公司Yup信用卡的市场机会与增长策略。通过分析世界银行宏观数据、Google Trends用户行为数据，结合广告投放历史表现，得出具体的商业洞察与战略建议。

## 项目背景

在代理Yup信用卡广告投放期间，观察到异常现象：
- Google渠道CPA仅为行业1/3，但Meta/TikTok效果差
- "食物素材"转化率高，传统金融素材效果差
- 猜测Yup服务于"被传统银行拒绝"的中等收入人群

本项目通过数据分析验证这些假设，并给出具体的增长建议。

## 技术栈

- **编程语言**: Python 3
- **数据分析**: Pandas, NumPy
- **数据获取**: World Bank(&API), Google Trends
- **数据可视化**: Plotly, Matplotlib
- **环境**: Google Colab
- **辅助工具**: AI对话助手（用于代码指导、思路启发）


## 核心分析内容

### 1. 宏观市场分析
- 金融包容性缺口：印尼31.7%中等教育人群无正规账户
- 互联网普及率：9年间从22%增长到69%
- 私人信贷/GDP对比：印尼38.7% vs 邻国52-160%
- 收入分配分析：60%人口为中低收入群体

### 2. 用户行为分析
- Google Trends搜索趋势对比
- BNPL竞争格局演变
- Yup市场定位验证

### 3. 战略建议
- SWOT分析矩阵
- 四大增长策略（SO/WO/ST/WT）
- 具体执行计划与预算分配
